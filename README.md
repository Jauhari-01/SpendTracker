## SpendTracker
MiniProject(Android base project)


Instructions : (For clone this project into your device)
    --> Install node.js
    --> Install Expo cli
    --> clone this repo
    --> Rum command : 
        npm install
        
        this will make this project run on your system.
        for more information you can follow official documentation.
